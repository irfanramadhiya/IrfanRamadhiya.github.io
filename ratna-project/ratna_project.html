Project_Ratna

PC Game / 2020
About This Project

Project_Ratna is a 2D platformer game for PC that was made as an assignment for BINUS Game Development Club (BGDC), one of the main uniqueness of this game is that players can reverse the gravity of the main character to solve challenging puzzles, this game demo was made by 5 people: Me along with another person as programmers who's in charge of programming the game, a game designer who's in charge of the game's ideas and development, an art designer who's in charge of making the pixel arts for the game, and a sound designer who's in charge of handling background music and sound fx.
