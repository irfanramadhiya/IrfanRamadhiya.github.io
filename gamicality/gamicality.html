Gamicality

Android Mobile App / 2021
About This Project

Gamicality is an Android app made for a college assignment, the main idea of the app is a forum app for gamers to interact with one another using posts and comments on a specific game of choice.

This was a 5 person project with my classmates, the main features of this app include: login, register, post, comment, and bookmark. Although the idea was to make it an online forum, for this project we only use local database to store data such as bookmarks and posts.
