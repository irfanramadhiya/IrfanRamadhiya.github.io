Resepin

Android Mobile App / 2022
About This Project

Resepin is an Android app made for a thesis project, the main functionality of the app is to search for recipes and order specific ingredients that are used within that recipe.

This was a 3 person project alongside 2 of my college friends, the main features of this app include: login, register, search recipe, create recipe, order ingredients, rate recipe, and we also added a working payment gateway using DOKU Sandbox.
